Defines a mail server usable to doorpi. (ex. send email if no one answers the doorbell)


For starters, get your mail outgoing server (SMTP) credentials

To define new account values use the "New Config Key" button. If you want to change already entered keys use "Edit Config Key".


Example Configuration for T-online mail service with the identifier bob

Edit New Config Key >>> Mail identifier bob [ENTER] >>> Keyname (as described below)

username
bob_wentnuts@t-online.de
password
password
email
bob_wentnuts@t-online.de
server
securesmtp.t-online.de
port
465
ssl
true
receiver
If this is left empty (default) your email is considered the receiver. If you want to send to a range of ppl, enter a mail address and hit "+" to enter another one.

The action is then defined with

Edit New Config Key >>> [Section Name] events >>> [Eventname] OnCallUnanswered (as an example) >>> [Action] mailto >>> [command] bob,door,missed a visitor,true


the command structure: mail_identifier, subject, text, send_snapshot

it is possible to send the content of a file. For that define text as /path/to/file.txt


Be aware that you need to take a snapshot (OnKeyDown,...) beforehand to send a timely picture
